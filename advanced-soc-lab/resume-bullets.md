## Project Title

MITRE ATT&CK-Based Detection Engineering and Automated SOC Response Lab

## Resume Bullets

- Built a MITRE ATT&CK-based detection engineering lab using Microsoft Sentinel, FortiGate, FortiAnalyzer, Windows Security Events, and KQL to detect brute-force activity, port scanning, suspicious PowerShell execution, privilege escalation, and RDP lateral movement.

- Developed and validated custom Sentinel analytics rules mapped to MITRE ATT&CK techniques including T1110, T1059.001, T1046, and T1021.001.

- Created a SOAR-style Logic Apps playbook to enrich Sentinel incidents with IP reputation context and add automated investigation comments to incident records.

- Produced SOC investigation documentation including detection logic, evidence screenshots, incident timelines, false-positive tuning notes, and response recommendations.
