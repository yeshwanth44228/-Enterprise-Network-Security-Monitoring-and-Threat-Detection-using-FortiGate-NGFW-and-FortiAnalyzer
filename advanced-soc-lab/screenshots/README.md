# Screenshots Folder

Place project screenshots in this folder using the names from `../docs/screenshot-checklist.md`.

Recommended naming pattern:

```text
01-resource-group.png
02-log-analytics-workspace.png
03-sentinel-enabled.png
...
23-final-github-repo.png
```

When screenshots are added, update `../docs/report-template.md` with image links and explanations.
